🚀 Cách chạy Django locally (trên máy tính)

Bước 1: Clone repository

git clone https://github.com/tranhuyhoang9bltk-art/ltud-topic10.git cd ltud-topic10

Bước 2: Cài đặt dependencies

pip install -r requirements.txt

Bước 3: Migrate database

python manage.py migrate

Bước 4: Tạo tài khoản admin

python manage.py createsuperuser

Bước 5: Chạy server

python manage.py runserver

✅ Truy cập tại: http://localhost:8000

📊 Admin panel tại: http://localhost:8000/admin
